Parece que o erro ocorreu porque o executável do Microsoft Edge WebDriver não foi encontrado no PATH do sistema. Para corrigir esse problema, você precisa baixar o Microsoft Edge WebDriver e configurar corretamente o PATH.

Aqui estão as etapas que você pode seguir para resolver o problema:

Certifique-se de ter baixado o Microsoft Edge WebDriver compatível com a versão do Microsoft Edge instalada no seu sistema. Você pode baixá-lo no seguinte link: http://go.microsoft.com/fwlink/?LinkId=619687

Após o download, extraia o arquivo ZIP para obter o executável MicrosoftWebDriver.exe.

Copie o arquivo MicrosoftWebDriver.exe para um local conhecido no seu sistema.

Abra o Painel de Controle do Windows e vá para "Sistema e Segurança" -> "Sistema" -> "Configurações avançadas do sistema".

Na guia "Avançado", clique no botão "Variáveis de Ambiente".

Na seção "Variáveis do Sistema", encontre a variável "PATH" e clique em "Editar".

Na janela "Editar Variável de Ambiente", clique em "Novo" e adicione o caminho completo para o diretório onde você copiou o arquivo MicrosoftWebDriver.exe. Por exemplo, se você o copiou para "C:\WebDriver", adicione "C:\WebDriver" como um novo caminho.

Clique em "OK" para fechar todas as janelas.

Depois de configurar corretamente o PATH, o erro deve ser resolvido e o código deverá ser capaz de encontrar o executável do Microsoft Edge WebDriver. Certifique-se de também ter as dependências corretas instaladas, como o pacote msedge-selenium-tools, que fornece a integração do Selenium com o Microsoft Edge WebDriver.

Peço desculpas pela informação anterior. Se o seu colega conseguiu executar o código sem configurar o PATH do Microsoft Edge WebDriver, é possível que ele tenha copiado o executável MicrosoftWebDriver.exe para um diretório que já está incluído no PATH do sistema.

Nesse caso, você pode seguir as etapas a seguir para verificar se o executável já está em um local acessível pelo sistema:

Certifique-se de ter baixado o Microsoft Edge WebDriver compatível com a versão do Microsoft Edge instalada no seu sistema.

Após o download, extraia o arquivo ZIP para obter o executável MicrosoftWebDriver.exe.

Verifique se você possui um diretório comum para armazenar executáveis, como C:\Windows ou C:\Windows\System32. Esses diretórios geralmente estão incluídos no PATH do sistema.

Copie o arquivo MicrosoftWebDriver.exe para o diretório escolhido.

Tente executar o código novamente e veja se o erro persiste.

Lembrando que essa abordagem depende do executável estar em um diretório já incluído no PATH do sistema. Se isso não funcionar, será necessário configurar o PATH como mencionado anteriormente.
